BullBank - Acessibilidade e Design Moderno
O BullBank é um ecossistema bancário digital que une tecnologia de ponta com princípios rigorosos de Design Inclusivo e Acessibilidade (A11y). O projeto foi desenvolvido para oferecer uma experiência fluida, moderna e totalmente adaptável a diferentes usuários e dispositivos.

🎨 Design e Identidade Visual
Todo o conceito visual do projeto foi idealizado e executado por mim, unindo fundamentos de design à funcionalidade técnica:

Logotipo Autoral: Criei o logotipo utilizando o conceito do trevo para simbolizar prosperidade e o "Bull" para representar a força do mercado financeiro.

Estética Glassmorphism: Utilização de transparências e borrões para profundidade visual, mantendo a sofisticação e a modernidade.

Fundamentos de Design: Aplicação de leitura constante e hierarquia tipográfica, garantindo que o usuário identifique rapidamente os campos de interação.



♿ Foco em Acessibilidade (A11y)
Este projeto foi construído para ser utilizável por todos:

Feedback em Tempo Real: Utilização do id="mensagem" para comunicar instantaneamente sucessos, erros ou alertas de segurança.

Contraste Otimizado: Escolha de cores (verde neon sobre fundo escuro) que respeitam os padrões de legibilidade.

Navegação Semântica: HTML estruturado para garantir que leitores de tela guiem usuários com deficiência visual de forma eficiente.

🚀 Funcionalidades Técnicas
Saudações Dinâmicas: O sistema identifica o horário local e saúda o usuário com "Bom dia", "Boa tarde" ou alertas sobre "Operações noturnas".

Autenticação Inteligente: Lógica em JavaScript que valida acessos específicos (como Ana Vitória ou Administrador) e emite alertas sobre possíveis invasões.

Simulação de Carregamento: Tela de loader personalizada para garantir que todos os elementos visuais e de segurança estejam prontos antes do acesso.

Responsividade Total: Uso de Media Queries para que a interface se adapte perfeitamente de celulares a monitores de alta resolução.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica e acessível.

CSS3: Estilização avançada com Flexbox e Media Queries.

JavaScript Vanilla: Manipulação do DOM, , switch/case e timers.
